# How to
1. Add it in your root build.gradle at the end of repositories:
```gradle
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
2. Step 2. Add the dependency build.gradle(.app)
```gradle
	dependencies {
	        implementation 'com.github.f2janyway:custom_view:Tag'
	}
```
